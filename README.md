# Design and Performance Analysis of an Electric Motor Drive System for a Four-Seater Electric Golf Cart

## 📌 Project Overview

This project focuses on the **design, modeling, analysis, and simulation of an electric motor drive system** for a four-seater electric golf cart.
The objective of this project is to develop an efficient, reliable, and cost-effective electric propulsion system using modern electric vehicle technologies.

The project includes:

* Motor selection and performance analysis
* Controller design and evaluation
* Battery and transmission calculations
* MATLAB / Simulink simulation
* Vehicle dynamics modeling
* Performance evaluation of the complete drive system

This work was carried out as a **Major Project** for the degree of **Bachelor of Technology in Electrical & Electronics Engineering**.

## 🔧 System Description

The complete electric drive system consists of:

### 1. Electric Motor

* BLDC motor used for propulsion
* Torque–speed analysis performed
* Power requirement calculated

### 2. Battery Pack

* Provides DC supply to controller
* Internal resistance considered
* State of charge effects studied

### 3. Motor Controller

* PWM / control logic used
* Controls speed and torque
* Implemented in MATLAB / Simulink

### 4. Vehicle Model

* Includes mass, drag, rolling resistance
* Calculates acceleration and speed

### 5. Simulink Model

Simulation includes:

* Motor block
* Battery subsystem
* Controller block
* Vehicle dynamics
* Load model
* Speed / torque scope
