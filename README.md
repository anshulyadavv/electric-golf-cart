# Design and Performance Analysis of an Electric Motor Drive System for a Four-Seater Electric Golf Cart

## 📌 Project Overview

This project focuses on the design, analysis, and simulation of an electric motor drive system for a four-seater electric golf cart.  
The goal of the project is to develop an efficient, reliable, and cost-effective electric propulsion system using modern electric vehicle technologies.

The project includes:

- Motor selection and analysis
- Controller analysis
- Battery and transmission calculations
- MATLAB/Simulink simulation
- Performance evaluation

This project was submitted as a Major Project for the degree of Bachelor of Technology in Electrical & Electronics Engineering.

## 📈 Key Results

- Selected Motor: 2 kW BLDC Motor
- Controller: Programmable PWM Controller
- Battery: 48V Pack
- Vehicle Range: ~50 km
- Maximum Speed: 25 km/h

The design satisfies the required performance for a four-seater electric golf cart.
